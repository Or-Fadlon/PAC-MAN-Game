About Page:

This Website Game developed by Elran Oren (205965908) & Orel Fadlon (316394717).

We used jQuery Plugin to validate the details entered by the user, those validations took place in the registration screen in the game.

We used Tamplates from googleapis and w3schools.

Difficulties encountered during the project were:
1. Move between the pages of the website, and save the details from previous pages.
2. Cause the ghosts to move on the game page screen in the way they will try to reach To Pacman in an intelligent way. at the end, we used BFS.
3. Difficulty using Object-oriented programming in javascript. We used the script type "module" to achive that difficulty.
